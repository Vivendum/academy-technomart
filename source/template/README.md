Папка для файлов разметки страниц.

--

Пример структуры файлов:

template/
├── head/
    ├── favicon.html
    └── другие *.html файлы содержащие служебную информацию
├── section/
    ├── header.html
    └── другие *.html файлы размечающие секцию
├── index.html
└── другие *.html файлы размечающие основу страницы
